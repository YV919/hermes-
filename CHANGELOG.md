# 更新日志

所有版本的更新内容记录。

## 1.0.0 (2026-06-18)

### 🚀 新功能

- 交互式配置 Hermes 使用 DMXAPI：依次填写配置名、base_url、API 密钥、模型、上下文窗口
- API 连接校验：保存前验证 base_url 与密钥是否可用
- 多套命名配置保存与一键切换，随时在不同配置间切换
- 写入前自动备份 Hermes 配置文件，避免误改丢失
- 环境自检：检查 Hermes 能否正常运行
- 非交互命令行模式：支持 `--list`（查看配置）、`--switch`（切换配置）、`--version`（查看版本）
- 跨平台支持：Windows / Linux / macOS
